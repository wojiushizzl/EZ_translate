# EZ_translate
# 一个超级简洁（陋）的桌面截图翻译助手

使用DEMO.gif

## 安装
git clone 
pip install -r requirements.txt

## 使用方法

##### python main.py
##### “Alt+z" 唤醒英译中
##### “Alt+x" 唤醒中译英
##### “Esc" 取消唤醒

## 环境（可以先跳过试试）
##### python == 3.8 
##### torch-2.1.1
##### torchaudio-2.1.1
##### torchvision-0.16.1
##### cuda_11.8.0
